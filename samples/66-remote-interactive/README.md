# Remote Agent - Interactive Operation Management

This example demonstrates interactive command-line control using a remote agent.

## Features

- Interactive CLI for managing operations


## Prerequisites

- Server agent running on `http://localhost:3500` (see example 65)

## Running the Example

First, start the server in another terminal:
```bash
cd ../65-simple-server-agent-no-execute-fn
go run main.go
```

Then run this interactive client:
```bash
go run main.go
```
