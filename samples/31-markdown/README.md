# Markdown Display Example

This example demonstrates how to use the `display.Markdown()` function to render formatted markdown content in the terminal with colors and styling.

## Usage

```bash
go run main.go
```

## What it demonstrates

- Headers (H1, H2, H3)
- Inline formatting: **bold**, *italic*, ~~strikethrough~~, `code`
- Unordered and ordered lists (with nesting support)
- Task lists (checked/unchecked)
- Code blocks with language syntax
- Blockquotes
- Links and images
- Horizontal rules
- Color-coded output

## Key Features

The `display.Markdown()` function:
- Uses only standard Go libraries (no external dependencies)
- Supports common markdown syntax
- Provides colorful terminal output
- Handles nested lists
- Preserves formatting and indentation
