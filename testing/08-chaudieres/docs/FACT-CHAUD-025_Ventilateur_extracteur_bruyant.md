## FACT-CHAUD-025: Ventilateur extracteur bruyant

| Champ | Valeur |
|-------|--------|
| **ID** | FACT-CHAUD-025 |
| **Catégorie** | Circuit fumées & tirage |
| **Système** | Ventilateur extraction |
| **Gravité** | Moyenne |
| **Marques** | Multi-marques |

**Symptômes :**
- Bruit anormal du ventilateur (vibration, sifflement, grincement)
- Bruit qui augmente avec le temps
- Vibration de la chaudière
- Performance dégradée (tirage insuffisant)

**Cause racine probable :**
Roulements usés, pales encrassées ou déséquilibrées, fixation desserrée, corps étranger, accumulation poussières, ventilateur en fin de vie.

**Étapes de résolution :**

1. **Identification type de bruit**
   - **Sifflement aigu** : fuite air, pale frottante
   - **Grincement** : roulement sec ou usé
   - **Vibration** : fixation, déséquilibre
   - **Claquement** : corps étranger, pale cassée

2. **Contrôle visuel et nettoyage**
   - Arrêter chaudière, déconnecter électriquement
   - Démonter capot accès ventilateur
   - Inspecter pales (propreté, état, déformation)
   - Nettoyer pales et corps ventilateur (soufflette, chiffon)
   - Vérifier absence corps étranger

3. **Contrôle mécanique**
   - Tourner ventilateur manuellement (doit être libre)
   - Sentir jeu axe/roulements (pas de jeu excessif)
   - Vérifier fixation ventilateur (vis serrées)
   - Contrôler silentblocs anti-vibration

4. **Contrôle roulements**
   - Écouter en faisant tourner manuellement (pas de bruit anormal)
   - Si grincement : roulement usé
   - Lubrification possible sur certains modèles (huile fine, 2-3 gouttes)
   - Remplacement ventilateur si roulement HS

5. **Contrôle équilibrage**
   - Pales doivent être identiques et propres
   - Encrassement asymétrique = déséquilibre = vibration
   - Nettoyer uniformément toutes les pales
   - Remplacer si pale cassée ou déformée

6. **Contrôle installation**
   - Vérifier fixation murale chaudière (bien rigide)
   - Installer plots anti-vibration si nécessaire
   - Isoler acoustiquement local si très bruyant

7. **Remplacement si nécessaire**
   - Durée vie ventilateur : 10-15 ans
   - Si roulements usés : remplacement complet recommandé
   - Commander référence exacte selon modèle chaudière
   - Prévoir joint ventilateur neuf

**Prévention :**
- Nettoyage annuel ventilateur
- Vérification fixations
- Lubrification si prévu constructeur
- Remplacement préventif après 12-15 ans

**Coût remplacement :**
- Ventilateur : 150-350 € selon modèle
- Main d'œuvre : 1-2 heures
- Prévoir référence exacte (étiquette ventilateur)
