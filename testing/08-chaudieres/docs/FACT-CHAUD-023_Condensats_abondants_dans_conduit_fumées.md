## FACT-CHAUD-023: Condensats abondants dans conduit fumées

| Champ | Valeur |
|-------|--------|
| **ID** | FACT-CHAUD-023 |
| **Catégorie** | Circuit fumées & tirage |
| **Système** | Évacuation condensats |
| **Gravité** | Moyenne |
| **Marques** | Multi-marques (chaudières condensation) |

**Symptômes :**
- Écoulement eau abondant du conduit de fumées
- Humidité autour du conduit
- Taches sur murs/plafond
- Corrosion conduit fumées
- Blocage évacuation condensats

**Cause racine probable :**
Condensation normale sur chaudière à condensation, conduit fumées non adapté (sans évacuation condensats), tubage non étanche, absence de siphon, pente inadaptée.

**Étapes de résolution :**

1. **Identification origine**
   - Chaudière condensation : condensation normale et voulue
   - Chaudière standard : condensation anormale (voir FACT-CHAUD-019)
   - Quantité normale condensats : 0,5-1 L/h selon puissance

2. **Contrôle conduit fumées**
   - Vérifier type conduit : doit être résistant condensats (inox, PP, PVC)
   - Conduit maçonné ancien : incompatible, destruction rapide
   - Tubage obligatoire : inox 316L ou plastique résistant

3. **Contrôle évacuation condensats**
   - Le conduit doit avoir pente vers chaudière ou collecteur
   - Vérifier présence siphon de neutralisation si requis
   - Contrôler évacuation vers égout ou chaudière
   - Nettoyer siphon si bouché

4. **Installation conforme**
   - **Conduit vertical** : pente vers chaudière, té de purge en bas
   - **Conduit horizontal** : pente 3% vers chaudière
   - Collecteur condensats tous les 3 m si nécessaire
   - Évacuation vers siphon chaudière ou neutralisateur

5. **Neutralisation condensats**
   - pH condensats : 3-4 (acide)
   - Neutralisation obligatoire si P > 25 kW (réglementation)
   - Kit neutralisation : granulés calcaires (neutralise acidité)
   - Remplacement granulés annuel

6. **Tubage conduit existant**
   - Si conduit maçonné : tubage obligatoire
   - Tubage flexible inox 316L ou rigide
   - Tubage concentrique si ventouse
   - Respect distances sécurité

7. **Contrôle étanchéité**
   - Vérifier assemblages (pas de fuite condensats)
   - Joints silicone haute température si nécessaire
   - Test étanchéité après installation

**Prévention :**
- Installation conforme dès origine
- Tubage adapté chaudière condensation
- Entretien annuel siphon et neutralisateur
- Vérification pentes et évacuation

**Réglementation :**
- DTU 24.1 : règles installation conduits
- Tubage obligatoire conduits maçonnés anciens
- Neutralisation si P > 25 kW
- Évacuation conforme assainissement
