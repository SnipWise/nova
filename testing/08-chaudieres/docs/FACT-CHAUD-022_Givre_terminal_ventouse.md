## FACT-CHAUD-022: Givre terminal ventouse

| Champ | Valeur |
|-------|--------|
| **ID** | FACT-CHAUD-022 |
| **Catégorie** | Circuit fumées & tirage |
| **Système** | Terminal ventouse |
| **Gravité** | **Élevée** |
| **Marques** | Multi-marques (chaudières ventouse) |

**Symptômes :**
- Formation de givre/glace au terminal ventouse
- Blocage chaudière sur défaut ventilateur ou pressostat air
- Obstruction partielle ou totale de l'évacuation
- Code erreur type F33 (Saunier Duval), F75 (Vaillant)
- Problème par temps froid et humide

**Cause racine probable :**
Condensation fumées qui gèle au terminal, terminal mal orienté (exposition vent/pluie), terminal non adapté (sans protection anti-givre), débit fumées insuffisant, terminal horizontal trop long.

**Étapes de résolution :**

1. **Dégivrage immédiat**
   - Arrêter la chaudière
   - Dégivrer le terminal (eau tiède, sèche-cheveux)
   - NE JAMAIS utiliser flamme directe
   - Vérifier libération complète passage air/fumées

2. **Diagnostic terminal**
   - Vérifier type de terminal (concentrique, séparé)
   - Contrôler orientation (idéal : vers le bas à 45°)
   - Mesurer longueur conduit horizontal (< 3 m recommandé)
   - Vérifier pente conduit (évacuation condensats)

3. **Contrôle condensation**
   - Sur chaudière condensation : condensation normale
   - Vérifier évacuation condensats en interne (siphon)
   - Si condensats s'accumulent dans conduit → givre
   - Améliorer pente évacuation

4. **Solutions anti-givre**
   - **Terminal rallongé** : éloigne zone condensation
   - **Terminal anti-givre** : avec réchauffage air
   - Modèles Ubbink, Chappée, Vaillant anti-givre
   - Coût : 80-150 €

5. **Modification installation**
   - Réorienter terminal (vers bas recommandé)
   - Réduire partie horizontale si trop longue
   - Améliorer isolation conduit (évite froid extrême)
   - Installer protection pluie/neige

6. **Réglages chaudière**
   - Augmenter température départ (moins de condensation)
   - Vérifier vitesse ventilateur (débit fumées suffisant)
   - Ajuster puissance minimale

**Prévention :**
- Installation terminal adapté climat (zones froides)
- Orientation optimale dès installation
- Vérification avant saison hivernale
- Surveillance par temps froid

**Recommandations installation :**
- Terminal orienté vers le bas à 45° (idéal)
- Longueur horizontale < 3 m
- Pente 3% vers chaudière (évacuation condensats)
- Terminal anti-givre si région froide
