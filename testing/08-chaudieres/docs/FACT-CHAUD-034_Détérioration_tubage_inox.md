## FACT-CHAUD-034: Détérioration tubage inox

| Champ | Valeur |
|-------|--------|
| **ID** | FACT-CHAUD-034 |
| **Catégorie** | Circuit fumées & tirage |
| **Système** | Tubage conduit |
| **Gravité** | **Élevée** |
| **Marques** | Multi-marques |

**Symptômes :**
- Corrosion visible sur tubage inox
- Perforation tubage (trou, déchirure)
- Condensats qui coulent à l'extérieur du tubage
- Taches humides sur conduit maçonné
- Diminution performances (fuite fumées)

**Cause racine probable :**
Tubage inox non adapté (304 au lieu de 316L), condensats acides (pH 3-4), mauvaise installation (assemblage, étanchéité), température fumées trop basse, vieillissement.

**Étapes de résolution :**

1. **Diagnostic type tubage**
   - Vérifier référence inox : **316L obligatoire** pour condensation
   - Inox 304 : inadapté, corrosion rapide
   - Identifier marquage sur tubage (gravé ou étiquette)
   - Durée de vie : 316L = 15-20 ans, 304 = 2-5 ans en condensation

2. **Évaluation dommages**
   - Inspecter visuellement zones accessibles
   - Localiser perforations (fumées, traces condensats)
   - Évaluer étendue corrosion
   - Si perforation : remplacement obligatoire

3. **Vérification installation**
   - Assemblages manchons : étanches ?
   - Raccordement bas (collecteur) : étanche ?
   - Raccordement haut (chapeau) : conforme ?
   - Pente correcte (évacuation condensats)

4. **Contrôle type de chaudière**
   - Chaudière condensation : 316L obligatoire
   - Chaudière basse température : 316L recommandé
   - Chaudière standard : 304 acceptable (si pas de condensation)

5. **Remplacement tubage**
   - Tubage flexible inox 316L ou rigide
   - Diamètre adapté selon puissance
   - Longueur = hauteur conduit + marge
   - Accessoires : collecteur, chapeau, manchons
   - Coût : 400-1000 € selon hauteur

6. **Installation correcte**
   - Ramonage préalable conduit existant
   - Insertion tubage (depuis toiture généralement)
   - Assemblage manchons avec silicone HT
   - Collecteur condensats en bas (étanche)
   - Chapeau terminal en haut
   - Centrage tubage dans conduit (équerres)
   - Isolation entre tubage et conduit si requis (DTU)

7. **Vérification conformité**
   - Test étanchéité (fumigène)
   - Contrôle tirage
   - Évacuation condensats fonctionnelle
   - Attestation installateur

**Prévention :**
- Tubage 316L uniquement pour condensation
- Vérification conformité matériaux à l'installation
- Inspection visuelle tous les 5 ans
- Remplacement préventif si corrosion débutante

**Réglementation :**
- DTU 24.1 : matériaux résistants condensats obligatoires
- Inox 316L ou équivalent (PP, PVC spécial)
- Responsabilité installateur si matériaux inadaptés
- Garantie décennale si vice caché

**Coût tubage :**
- Flexible inox 316L : 15-25 €/m
- Rigide inox 316L : 25-40 €/m
- Collecteur : 80-150 €
- Chapeau : 60-120 €
- Main d'œuvre : 400-800 € (selon hauteur)
